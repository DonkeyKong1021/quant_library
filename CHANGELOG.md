## [0.1.0] - 2026-01-19

### Added
- Initial release of QuantLib
- Data fetching from Yahoo Finance
- Backtesting engine with event-driven simulation
- Technical indicators: SMA, EMA, RSI, Bollinger Bands, MACD, and more
- Risk metrics: Sharpe ratio, Sortino ratio, VaR, drawdown calculations
- Portfolio management: position sizing, rebalancing
- Visualization tools for performance analysis
- Strategy framework with base class
- PostgreSQL database storage
- FastAPI REST API
- React web application
- Streamlit dashboard interface
- Example strategies and usage patterns
